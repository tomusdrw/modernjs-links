Jak zacząć?

Przechodzimy do katalogu projektu.

      $ npm init -f # tworzymy package.json

Instalujemy potrzebne zależności developerskie

      $ npm install browserify --save-dev  # Do bundlowania
      $ npm install watchify --save-dev    # Do obserwowania
      $ npm install live-server --save-dev # Do serwowania

Wrzucamy nasze pliki do `src`, aby dostać taką strukturę:

      |-- package.json
      |-- node_modules/
      `-- src
          |-- index.html
          `-- main.js

Odpalamy watchify:

      $ # To będzie produkować bundla ze wszystkich naszych modułów
      $ node_modules/.bin/watchify src/main.js src/bundle.js

Odpalamy serwer (w katalog `src`!):

      $ cd src
      $ ../node_modules/.bin/live-server

Upewniamy się, że w index.html ładujemy tylko pliku "bundle.js"

      <script src="bundle.js"></script>

Od teraz możemy korzystać z modułów CommonJS w naszej aplikacji
